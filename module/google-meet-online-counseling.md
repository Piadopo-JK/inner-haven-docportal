# Google Meet / Online Counseling

[Project Homepage](project-homepage.md) > Google Meet / Online Counseling

---

## Functional Description

The system must allow counseling sessions to be conducted online through Google Meet integration. When an appointment is approved for online mode, a meeting link must be generated or attached to the appointment.

---

## Use Case Scenario

| Actor | Student / Counselor |
|-------|---------------------|
| Precondition | Appointment is approved and set to online mode |
| Main Flow | System generates meeting link → Link attached to appointment → Users join session |
| Alternative Flow | Link generation fails → Display error |
| Postcondition | Online counseling session conducted successfully |
