# Online Appointment Scheduling

[Project Homepage](project-homepage.md) > Online Appointment Scheduling

---

## Functional Description

The system must allow students to book counseling appointments by selecting a preferred date, time, and counselor. The appointment request is stored and forwarded to the counselor for approval.

---

## Use Case Scenario

| Actor | Student |
|-------|---------|
| Precondition | Student is logged in |
| Main Flow | Student selects counselor → Selects time slot → Submits request |
| Alternative Flow | No available slots → Display error |
| Postcondition | Appointment request recorded |
