# Secure Login & Authentication

[Project Homepage](project-homepage.md) > Secure Login & Authentication

---

## Functional Description

The system must implement secure authentication for both students and counselors to protect sensitive information. Users must log in using registered credentials before accessing system features.

---

## Use Case Scenario

| Actor | Student / Counselor |
|-------|---------------------|
| Precondition | User has a registered account |
| Main Flow | User enters credentials → System validates → Access granted |
| Alternative Flow | Invalid credentials → Error message displayed |
| Postcondition | User session established securely |
