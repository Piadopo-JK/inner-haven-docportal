# Counselor Notes / Feedback

[Project Homepage](project-homepage.md) > Counselor Notes / Feedback

---

## Functional Description

The system must allow counselors to record notes, recommendations, and follow-up instructions after a counseling session. Notes must be securely stored and accessible only to authorized counselors.

---

## Use Case Scenario

| Actor | Counselor |
|-------|------------|
| Precondition | Counseling session completed |
| Main Flow | Counselor inputs session notes → Saves record |
| Alternative Flow | Save error → Prompt retry |
| Postcondition | Notes securely stored for future reference |
