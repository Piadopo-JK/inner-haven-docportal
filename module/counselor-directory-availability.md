# Counselor Directory & Availability

[Project Homepage](project-homepage.md) > Counselor Directory & Availability

---

## Functional Description

The system must display a list of available counselors including their specialization, office location, and available time slots. Students should be able to view open schedules before booking an appointment.

---

## Use Case Scenario

| Actor | Student |
|-------|---------|
| Precondition | Student is logged in |
| Main Flow | Student views counselor directory → Selects counselor → Views available time slots |
| Alternative Flow | No availability found → Display message |
| Postcondition | Student selects preferred schedule |
