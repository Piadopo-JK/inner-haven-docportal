# Anonymous Help Request

[Project Homepage](project-homepage.md) > Anonymous Help Request

---

## Functional Description

The system must allow students to submit confidential help requests without revealing personal identity. No login is required for this feature, and submissions are forwarded privately to the guidance office.

---

## Use Case Scenario

| Actor | Student |
|-------|---------|
| Precondition | Student accesses help request page |
| Main Flow | Student submits anonymous message → System records request |
| Alternative Flow | Empty submission → Prompt validation error |
| Postcondition | Anonymous request stored securely |
